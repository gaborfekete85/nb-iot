# AlarmOn Application

## Requirements

- Node and npm
```
curl -sL https://deb.nodesource.com/setup_7.x | sudo -E bash -
sudo apt-get install -y nodejs
sudo apt-get install -y npm
```

- Tools you will need
```
sudo apt-get install git
sudo apt-get install curl
```
